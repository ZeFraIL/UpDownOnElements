# אפליקציית Up Down On Elements

## מטרת האפליקציה

אפליקציה זו משמשת כמונה פשוט הנשלט על ידי תנועת המכשיר. היא משתמשת בחיישן התאוצה (accelerometer) כדי לזהות תנועות הטיה.

-   **הטיה קדימה:** הטיית המכשיר קדימה (הטיה חיובית בציר ה-X) תגדיל את המספר המוצג על המסך.
-   **הטיה אחורה:** הטיית המכשיר אחורה (הטיה שלילית בציר ה-X) תקטין את המספר.

## MainActivity

`MainActivity` הוא המסך הראשי של האפליקציה.

-   הוא מאתחל את ה-`SensorManager` ורושם "מאזין" (listener) לחיישן התאוצה.
-   הוא מציג `TextView` המראה את הספירה הנוכחית.
-   הלוגיקה המרכזית נמצאת במתודה `onSensorChanged`, אשר מנטרת את נתוני חיישן התאוצה (`event.values[0]` עבור ציר ה-X) כדי לקבוע את כיוון ההטיה ולעדכן את המונה בהתאם.
-   נעשה שימוש בדגלים (`isCountingUp`, `isCountingDown`) כדי להבטיח שהמספר יוגדל או יוקטן פעם אחת בלבד עבור כל תנועת הטיה, מה שמונע שינויים מרובים מהטיה רציפה אחת. הדגלים מתאפסים כאשר המכשיר חוזר למצב ניטרלי יחסית.
