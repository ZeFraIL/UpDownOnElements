# Приложение Up Down On Elements

## Назначение приложения

Это приложение представляет собой простой счетчик, управляемый движением устройства. Оно использует датчик акселерометра для обнаружения наклонов.

-   **Наклон вперед:** Наклон устройства вперед (положительный наклон по оси X) увеличивает число, отображаемое на экране.
-   **Наклон назад:** Наклон устройства назад (отрицательный наклон по оси X) уменьшает число.

## MainActivity

`MainActivity` — это основной экран приложения.

-   Он инициализирует `SensorManager` и регистрирует слушателя для акселерометра.
-   Он отображает `TextView`, который показывает текущий счет.
-   Основная логика находится в методе `onSensorChanged`, который отслеживает данные акселерометра (`event.values[0]` для оси X) для определения направления наклона и соответствующего обновления счетчика.
-   Флаги (`isCountingUp`, `isCountingDown`) используются для того, чтобы число увеличивалось или уменьшалось только один раз за один жест наклона, предотвращая многократные изменения от одного непрерывного наклона. Флаги сбрасываются, когда устройство возвращается в относительно нейтральное положение.
